# Product Specification

## Primary Users

- College students decorating dorm rooms
- Young adults moving into apartments

## Core User Journey

1. Browse inspiration
2. Explore a room in 3D
3. Discover a Nook they love
4. Save that Nook
5. Import it into their own room
6. Customize placement
7. Purchase products

## Home Screen

- Social feed
- Search
- Tags
- My Projects

## Room Page

Interactive 3D room

Popup panel containing:

- Title
- Creator
- Description
- Tags
- Like
- Save
- Remix / Import
- Share

Separate tab:

- All products in room

Clicking an object inside the room highlights it in the product list.

## Profiles

Pinterest-style creator profiles

Collections

Uploaded rooms

Saved inspiration

## Projects

Users create their own rooms.

Instead of copying another room exactly, they import Nooks from many different rooms.

Users position every imported Nook inside their own layout.

## Why This Room Works

Each room can include annotations explaining why certain design choices work.
